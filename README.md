# Liquid Stone

Website one-page premium, responsive și fără dependențe externe.

## Demo online

https://nache5.github.io/liquid-stone-demo/

## Rulare locală

```bash
npm run dev
```

Apoi deschide `http://localhost:4173`.

## Structură

- `index.html` — structură, copy în română și SEO
- `styles.css` — design responsive și animații
- `script.js` — meniu mobil, reveal la scroll, slider și formular demo
- `assets/images/` — imaginile originale și versiunile optimizate

## Înainte de publicare

Înlocuiește numărul de telefon, adresa de email, linkurile sociale și conectează formularul la serviciul de trimitere preferat.
